默认提供两个模板

  • cra-template
  • cra-template-typescript

自定义模板

Custom Templates | Create React App

自定义模板包命名规则

带范围的模板命名：@[scope-name]/cra-template 或者 @[scope-name]/cra-template-[template-name]

分别对应：@[scope] 和 @[scope]/[template-name]
不带范围的：cra-template-[template-name]
